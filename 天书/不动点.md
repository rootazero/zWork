# 不动点

> *"凡试图定义自身的定义必导向无定义，除非它定义自身为不可定义。"*
>
> ——摩西·迈蒙尼德，《困惑者指南》第二卷第七章附注（1190年），萨拉丁王朝

---

**定理陈述**

第三十七年，秋分日，我在卡珊德拉以西73公里的一座废弃修道院遇到一位数学家。

修道院建于七百年前，用于抄写经文。现在只剩图书馆，藏书已腐烂成灰，留下空荡荡的书架和满地的虫蛀痕迹。数学家住在图书馆二层，墙上写满公式。

他正在证明一个定理：**关于天书的不动点定理**。

以下是他的笔记本的摘录。我保留了原有的符号系统和证明结构。

---

**笔记1：布劳威尔不动点定理**

设 D ⊂ ℝⁿ 是一个紧凸集，f: D → D 是连续映射。则f至少有一个不动点，即存在 x* ∈ D 使得 f(x*) = x*。

**证明**（拓扑学方法）：反证法。假设对所有 x ∈ D，f(x) ≠ x。则可定义连续映射 g: D → ∂D（D的边界），g(x) 为从f(x)出发沿直线指向x的射线与∂D的交点。这个映射是从D到∂D的缩并（retraction），即 g|∂D = id。但这与同调论矛盾：Hₙ(D) = 0 而 Hₙ₋₁(∂D) ≠ 0（对于n维单纯形）。因此假设不成立。Q.E.D.

**应用**：此定理保证了某些方程的解的存在性，即使我们无法显式求解。

---

**笔记2：巴拿赫不动点定理**

设 (X, d) 是完备度量空间，T: X → X 是压缩映射，即存在 λ ∈ [0,1) 使得对所有 x, y ∈ X：

d(T(x), T(y)) ≤ λ d(x, y)

则T有唯一不动点 x*，且对任意初值 x₀ ∈ X，迭代序列 xₙ₊₁ = T(xₙ) 收敛到 x*。

**证明**：定义序列 x₀, x₁ = T(x₀), x₂ = T(x₁), ...。则：

d(xₙ₊₁, xₙ) = d(T(xₙ), T(xₙ₋₁)) ≤ λ d(xₙ, xₙ₋₁) ≤ ... ≤ λⁿ d(x₁, x₀)

对于 m > n：
d(xₘ, xₙ) ≤ Σᵢ₌ₙᵐ⁻¹ d(xᵢ₊₁, xᵢ) ≤ d(x₁, x₀) Σᵢ₌ₙᵐ⁻¹ λⁱ = d(x₁, x₀) λⁿ/(1-λ) → 0

故{xₙ}是柯西列，由完备性收敛到某点 x* ∈ X。由T的连续性：T(x*) = T(lim xₙ) = lim T(xₙ) = lim xₙ₊₁ = x*。

唯一性：若y*也是不动点，则 d(x*, y*) = d(T(x*), T(y*)) ≤ λ d(x*, y*)，而λ < 1，故d(x*, y*) = 0。Q.E.D.

**应用**：这个定理不仅保证不动点存在，还提供了寻找它的算法（迭代）。

---

**笔记3：天书作为不动点问题**

设 𝔹 为所有可能的书的集合。定义映射 D: 𝔹 → 𝔹（"描述"映射）：对每本书 B，D(B) 是"关于B的完整描述"。

**问题**：天书是否是D的不动点？即，是否存在 T ∈ 𝔹 使得 D(T) = T？

换言之：是否存在一本书，它关于自身的完整描述恰好等于它本身？

**初步分析**：

(i) 如果 T = D(T)，则T包含对自身的完整描述。这意味着T包含T的内容、T的结构、T的元数据...包括"T包含对自身的完整描述"这个事实。这导致无穷回归。

(ii) 设 |B| 表示书B的长度（字符数）。则 |D(B)| ≥ |B|，因为描述一个对象至少需要原对象的信息量。更准确的说，如果描述是"完整"的，则 |D(B)| ≥ |B| + c，其中c是描述元数据的开销（"这本书长度为...", "包含以下内容..."等）。

因此：|D(T)| ≥ |T| + c。但如果 D(T) = T，则 |D(T)| = |T|，矛盾。

**结论（初步）**：在有限长度的书籍空间中，不存在D的不动点。

---

**笔记4：无限维情况**

也许天书是无限长的？

设 𝔹∞ 为所有可数无限长序列的空间。在适当的度量下（如 d(B, B') = Σₙ 2⁻ⁿ min(1, |Bₙ - B'ₙ|)，其中Bₙ是第n个字符），𝔹∞ 是完备度量空间。

重新定义 D: 𝔹∞ → 𝔹∞。

**问题**：D是压缩映射吗？

设 B 和 B' 的前k个字符相同，从第k+1个字符开始不同。则 D(B) 和 D(B') 在描述"前k个字符"的部分相同，但在描述"第k+1个字符"的部分不同。

如果描述是系统的（如："第n个字符是X"），则D(B)和D(B')的差异出现在大约第k个字符的位置（因为需要k个字符来描述"前k个字符"）。因此：

d(D(B), D(B')) ≈ 2⁻ᵏ ≈ d(B, B')

这意味着λ ≈ 1，不是压缩。巴拿赫定理不适用。

但布劳威尔定理呢？𝔹∞不是紧的（无限维），因此布劳威尔定理也不适用。

---

**笔记5：自指的代数**

换一个角度。设 T 为天书的"内容"（作为抽象的信息对象，独立于任何物理表示）。定义操作：

- D(T) = "T的描述"
- I(T) = "T本身"

如果天书是自我描述的，则：I(T) = D(T)

但这是什么意思？"T的描述"和"T本身"有什么区别？

在自然语言中：
- "战争与和平" ≠ "一本关于拿破仑时代俄国贵族的小说"（前者是书，后者是描述）

在形式语言中：
- 哥德尔编码：⌈φ⌉ ≠ φ（⌈φ⌉是公式φ的哥德尔数，φ是公式本身）

但也许天书的特殊性在于：它有能力包含自己的哥德尔编码？

**引理**：在算术系统中，任何公式φ都可以通过对角线化构造一个公式ψ，使得：

PA ⊢ ψ ↔ φ(⌈ψ⌉)

即，ψ"说"的是φ关于ψ自身的哥德尔数所说的话。

**应用**：设 Desc(x) 为"x是真的描述"的谓词。通过对角线化，存在公式T使得：

PA ⊢ T ↔ Desc(⌈T⌉)

这个T说："我的哥德尔数是一个真的描述。"

这类似于说谎者悖论的变体。如果T为真，则⌈T⌉确实是真的描述——但描述什么？描述T本身。那么T的内容是什么？是"我的哥德尔数是真的描述"。因此T的描述就是T本身。

**T是D的不动点！**

但代价是什么？代价是T在PA中不可判定——我们无法证明T为真或为假，除非在更强的系统中。

---

**笔记6：哥德尔式证明**

更精确的表述。

设L为一阶语言，T为L的理论（一组公理）。设Prov_T(x)为"x的哥德尔数在T中可证"的谓词。

**哥德尔第一不完备性定理**：如果T是一致的且包含足够的算术，则存在句子G使得：

T ⊬ G 且 T ⊬ ¬G

且G可以构造为：G ↔ ¬Prov_T(⌈G⌉)（"我不可证"）

现在考虑另一个构造：设Desc_T(x, y)为"x是y的完整描述（在理论T中）"的二元谓词。通过对角线化，存在句子H使得：

H ↔ ∃x [Desc_T(x, ⌈H⌉) ∧ x = ⌈H⌉]

即："存在我的描述，且这个描述等于我本身。"

**分析**：如果H为真，则存在x使得x既是H的描述又等于H的哥德尔数。但"描述"通常比"被描述者"更长（需要元语言）。因此x = ⌈H⌉意味着H的描述长度等于H本身——这只在H是自指的情况下可能。

更深的问题：H在T中的真值是什么？

**情况1**：如果T ⊢ H，则H在T的标准模型中为真。这意味着确实存在H的描述等于H本身。但这个"描述"在哪里？它必须在T的某个模型中存在——也许在非标准模型中，"长度"的概念被扩展，允许某种无穷的自包含结构。

**情况2**：如果T ⊬ H，则H在某些模型中为真、在某些模型中为假。在H为假的模型中，不存在H的自我描述。在H为真的模型中，存在——但我们在T内无法"看到"它。

**结论**：天书（作为自我描述的书）在标准算术模型中不存在，但在某些非标准模型中存在。

这类似于说：天书不在我们的宇宙中，而在某个更大的、包含非标准无穷大的宇宙中。

---

**笔记7：塔斯基不可定义性**

还有一个障碍：**塔斯基真理不可定义定理**。

**定理**（塔斯基）：在包含足够算术的理论T中，不存在谓词True_T(x)使得对所有句子φ：

T ⊢ True_T(⌈φ⌉) ↔ φ

即，真理谓词无法在语言内定义。

**应用于天书**：如果天书的"完整描述"要求列出天书的所有真理（"这本书包含...，这本书不包含..."），那么描述需要一个真理谓词。但塔斯基定理说这不可能。

因此，"关于天书的完整描述"在逻辑上不可能存在——除非这个描述在更高阶的元语言中。

设 L₀ 为对象语言（天书用 L₀ 写成），L₁ 为元语言（描述天书的语言），L₂ 为元元语言（描述L₁的语言），...

天书的完整描述需要在 L₁ 中。但如果天书本身需要包含这个描述，它必须包含 L₁ 的内容。但 L₁ 比 L₀ 更强（它有 L₀ 的真理谓词），因此天书必须用 L₁ 书写。

但那么天书的描述需要在 L₂ 中...无穷回归。

**结论**：天书的完整描述需要无穷多层语言。因此天书（如果存在）不在任何有限层的语言层级中，而在Lω——所有有限层的并集的极限。

但 Lω 是不可数的（如果每一层可数的话）。这意味着天书在某种意义上"超越可数性"。

---

**笔记8：范畴论视角**

最后一个尝试。在范畴论中考虑不动点。

设 𝒞 为"书籍"范畴，对象是书，态射是"引用"（一本书引用另一本书）。定义函子 D: 𝒞 → 𝒞，D(B) 是"关于B的书"。

**问题**：D是否有不动点对象，即存在T使得 T ≅ D(T)（同构）？

**劳弗尔定理**（Lawvere's fixed point theorem）：在笛卡尔闭范畴中，如果存在满态射 e: A → (A ⇒ B)（其中 A ⇒ B 是指数对象），则每个内射 f: B → B 都有不动点。

**应用**：如果存在"万有书" U 和满态射 e: U → (U ⇒ 𝔹)（即，U 能"表示"所有从U到书籍的映射），则描述函子必有不动点。

但"满态射"要求e是满射——每个函数 U → 𝔹 都能被U中的某个元素编码。这类似于要求U能表达"所有可能的关于自身的陈述"。

这又遭遇哥德尔障碍：没有足够强的系统能表达所有关于自身的真陈述。

因此：**除非在非常特殊的范畴中（如，允许非良基集合的范畴），否则不存在描述函子的不动点。**

---

**笔记9：唯一的不动点**

经过以上分析，我得出一个令人沮丧的结论：

**定理（天书的不存在性）**：在任何标准的数学框架中（一阶逻辑、ZFC集合论、标准算术），不存在书T使得"T的完整描述等于T本身"。

但等等。有一个例外。

考虑**空集** ∅。

空集的描述是："不包含任何元素的集合"。这个描述本身是什么？是一个陈述，一段文字。如果我们形式化这个描述为一个集合（如，字符的序列），它不等于∅。

但在另一个意义上：空集的"完整信息"是什么？是"无"。而"无"的描述也是"无"。从信息论角度，∅ 的信息量为零，描述∅ 的信息量也为零（你只需说"空"，这是常数大小的信息）。

因此：**∅ 在某种意义上是描述函子的不动点。**

还有一个例外：**全集** 𝕌（所有集合的集合，虽然在ZFC中它导致罗素悖论）。

𝕌 的描述是："包含一切的集合"。如果这个描述本身也是一个对象，它应该被包含在 𝕌 中。因此 𝕌 包含对自身的描述。在某种意义上，𝕌 = D(𝕌)。

但代价是：𝕌 在ZFC中不存在（罗素悖论）。它只在非良基集合论（如阿克塞尔集合论，ZFA）中存在。

**结论**：
1. **如果天书 = ∅（空书），它是不动点。但它什么都不包含。**
2. **如果天书 = 𝕌（全集），它是不动点。但它在标准数学中不一致。**

天书只能是两个极端之一：完全的空无，或完全的矛盾。

---

**笔记10：我的证明**

综合以上，我现在证明核心定理。

**定理**（天书的不动点性质）：设 𝔹 为书籍空间，D: 𝔹 → 𝔹 为"完整描述"映射。则：

(1) 在有限维、标准的数学框架中，D没有非平凡不动点。

(2) D的平凡不动点只有两个：∅（空书）和 𝕌（全书，若允许非良基集合）。

(3) 任何非平凡的不动点必然在某个扩展的数学框架中（非标准模型、无穷语言层级、非良基集合论等）。

(4) 但在这些扩展框架中，"完整描述"的定义本身变得不明确，因此不动点的存在性无法判定。

**证明**：(1)已在笔记3中证明。(2)已在笔记9中论证。(3)来自笔记6和7的哥德尔-塔斯基障碍。(4)是元数学的限制。Q.E.D.

**推论**（天书的悖论性）：如果天书被定义为"包含一切的书"（对应 𝕌），它在标准数学中导致矛盾。如果天书被定义为"关于自身的完整描述"，它要么不存在（在标准框架中），要么存在但不可判定（在扩展框架中）。

**因此：寻找天书等价于寻找逻辑系统的不动点——一个自洽且完备的系统。哥德尔已经证明这不可能。**

---

**尾声**

我读完笔记，问数学家："那你为什么还在寻找？"

他在黑板上写下最后一个公式：

**T = {x | x ∉ T}**

罗素悖论的标准形式。然后他改写：

**T = {x | x是关于T的陈述且x为假}**

"这是天书的定义，"他说。"天书包含所有关于天书的假陈述。"

"那关于天书的真陈述在哪里？"

"不在天书中。它们在外面，在我们关于天书的谈话中，在寻找的过程中。"

"所以天书包含一切，除了真相？"

"是的。或者说，天书就是真相的补集。寻找天书的过程——所有的失败、所有的误解、所有的错误的道路——这些都被包含在天书中。只有成功的那一刻，只有找到天书的那个瞬间，不在天书中。"

"那找到天书是不可能的？"

"不。找到天书是可能的，但一旦找到，天书就不再是天书——因为成功的寻找不被包含在'所有失败的寻找'的集合中。"

他指向墙上的一行字，倾斜90度：

**"唯一的不动点是运动本身。"**

我离开修道院时，数学家在笔记本最后一页写道：

**证明完毕。但证明什么都没有证明，因为天书不在任何证明的结论中——它在证明的过程中，在推理的每一步，在笔划的每一道墨迹中。我写了372页证明，试图固定天书的位置。但天书不是函数的值，而是函数本身。**

**D(T) = T 的唯一解是：T = D。**

**天书即是描述行为本身。**

**【编者按】**

此笔记本发现于废弃修道院，日期标注为"第三十七年"。作者不详。笔记本共372页，最后23页空白。

现代数理逻辑学家对此笔记的评价分歧：一些人认为这是对哥德尔不完备性定理的文学化演绎，有一定洞察但缺乏严格性；另一些人指出其中关于"描述函子的不动点"的讨论触及了范畴论中自指的核心问题，与劳弗尔定理有深刻联系。

关于天书与不动点的关系，数学上最接近的类比是**Y组合子**（不动点组合子）：在λ演算中，Y = λf.(λx.f(x x))(λx.f(x x)) 满足对任意函数g，Y g = g(Y g)。这个组合子"固定"了任何函数的不动点——即使函数本身没有显式定义的不动点。

如果天书对应某种"宇宙的Y组合子"，它不是一个静态对象，而是一个递归过程——不断将"关于自身的描述"应用到自身，无限展开，永不收敛。

也许这就是为什么所有寻找天书的人都在运动中。他们不是在接近一个目标，而是在执行一个无穷迭代。他们是Y组合子的参数，被应用、被展开、被递归，永无止境。

而数学已经证明：**有些函数的不动点只能通过无穷迭代逼近，永远无法在有限步内抵达。**

也许天书就是这样一个不动点。

**咿。**
