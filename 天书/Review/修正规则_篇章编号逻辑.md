# 篇章编号逻辑漏洞修正规则

## 问题描述

**设定矛盾**:
- 全书设定:正文是散落文本,由编者汇编
- 逻辑漏洞:正文中第一人称主人公知道"第X篇"编号

**示例问题**:
第20篇:"我想起第十一篇中的那个守望者" ← 主人公怎么知道编号?

---

## 修正方案C:选择性保留(折中)

### ✅ 可以保留"第X篇"的情况

1. **导航链接**(文档底部`[← 第X篇]`):✓ 保留
2. **元文档**(CLAUDE.md、Planning/、Review/等):✓ 保留
3. **注释**:
   - 有【编者注】标记的:✓ 保留
   - 正文注释引用:需加【编者注】标记
4. **元叙事篇**(16、17、18、19、24篇):
   - 这些本身就是编者视角
   - 可用"档案ASC-00X"或"第X篇"
   - ✓ 基本保留

### ❌ 必须修正的情况

**正文中第一人称主人公的直接引用**

#### 类型一:正文叙述中的引用

❌ 错误示例:
```
我想起第十一篇中的那个守望者
```

✅ 修正示例:
```
我想起那个终生走向地平线的守望者H
```

#### 类型二:正文脚注中的引用

❌ 错误示例:
```
^[5] 参见第14篇《沉默的语言》
```

✅ 修正示例:
```
^[5] 【编者注:参见第14篇《沉默的语言》】
```

#### 类型三:正文中的回溯性引用

❌ 错误示例:
```
*(那时我还不明白,这在第21篇才会揭示...)*
```

✅ 修正方案:
```
*(那时我还不明白,这在多年后才会揭示...)*
或
*【编者按:这在第21篇才会揭示...】*
```

## 具体修正策略

### 策略一:改为具体描述(优先)

用人物特征、事件特征替代编号:

| 编号引用 | 改为具体描述 |
|---------|------------|
| 第11篇的守望者 | 那个终生走向地平线的守望者H |
| 第14篇的沉默 | 那个研究沉默的语言学家 |
| 第21篇的园丁 | 那个在矛盾花园中浇水的园丁 |
| 第3篇的日记 | 索菲亚人的善行账簿/黑色皮革铜扣的账簿 |
| 第5篇的图书馆 | 那座遗忘的六边形图书馆 |
| 第6篇的时刻 | 第42年三月十七日的停止 |

### 策略二:加【编者注】(次选)

用于注释或括号性说明:

```
参见第14篇 → 【编者注:参见第14篇《沉默的语言》】
```

### 策略三:改为时间/地点引用

```
第7年的相遇 → 那次在阿尔法平原的相遇
第42年的决定 → 那个春分日正午的决定
```

3. ⬜ 逐篇检查上下文:

## 边界案例判断

### 案例A:第1篇注释中的跨篇引用

```原文
见第6篇《第42年》和第21篇《矛盾的园丁》
```

**判断**:这是注释,可加【编者注】
**修正**:
```
【编者注:见第6篇《第42年》和第21篇《矛盾的园丁》】
```

### 案例B:第20篇正文引用

```原文
我想起第十一篇中的那个守望者
```

**判断**:正文第一人称叙述,必须改
**修正**:
```
我想起那个终生走向地平线的守望者H
```

### 案例C:第16篇元叙事引用

```原文
文献D(ASC-005-001)
```

**判断**:元叙事篇编者视角,保留 ✓

## 注意事项

1. **保持叙事流畅**:改完后读一遍,确保自然
2. **保持风格一致**:用符合该篇风格的描述方式
3. **不要过度改动**:
   - 导航链接完全保留
   - 注释只加【编者注】标记即可
   - 只改必须改的地方
4. **特殊数字保留**:
   - "第42年""第7年"等时间标记保留(这是时间,不是篇章编号)
   - "第七个柜子""第三层"等空间标记保留
