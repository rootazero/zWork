## 四大核心技能

### 🎨 技能1: 博尔赫斯式风格 (style.md)

**何时激活**:
- 写作任何新段落时
- 修改已有文本的文风时
- 检查是否偏离博尔赫斯式精确性时

**核心能力**:
- 三种叙事层次的切换（诗性/哲学/元叙事）
- 博尔赫斯式精确性（数字/地名/不确定性）
- 冷峻与无力感的营造
- 节奏与质感的控制

**输出**:
- 精确但简练的语言
- 冷静但有张力的叙述
- 理性失败的无力感

**何时不用**:
- 规划结构时（用技能4）
- 查找世界观设定时（用技能2）

➡️ [详见 skills/style.md](skills/style.md)

---

### 🌍 技能2: 世界观设定 (worldbuilding.md)

**何时激活**:
- 涉及阿尔法族文字系统时
- 涉及索菲亚或卡珊德拉两座城市时
- 涉及天书的定位/形态时
- 需要检查设定一致性时

**核心能力**:
- 阿尔法文字的精确使用（0°/33°/90°/45°）
- 道德守恒方程的应用（W(x)+R(x)=K）
- 天书七种定位的选择
- 世界观铁律的遵守

**输出**:
- 符合设定的场景描写
- 精确的倾斜角度
- 守恒方程的正确使用
- 阿尔法族只能"咿""呀"

**何时不用**:
- 不涉及特定世界观元素时
- 纯哲学思辨段落（但仍需检查是否违背设定）

➡️ [详见 skills/worldbuilding.md](skills/worldbuilding.md)

---

### 🧠 技能3: 理论工具箱 (theory.md)

**何时激活**:
- 需要引入数学/物理/哲学理论时
- 构建核心悖论时
- 需要理论支撑思辨段落时
- 检查理论使用是否恰当时

**核心能力**:
- 罗素悖论/康托尔对角线等数学工具
- 量子叠加/相对论等物理工具
- 维特根斯坦/柏拉图等哲学框架
- 理论的文学化表达

**输出**:
- 严密但导向悖论的推理
- 文学化的理论呈现（故事优先）
- 理性失败的展示

**何时不用**:
- 纯诗性段落
- 不需要理论支撑的场景描写

➡️ [详见 skills/theory.md](skills/theory.md)

---

### 📐 技能4: 叙事结构 (structure.md)

**何时激活**:
- 开始写新篇之前（规划结构）
- 选择开篇/结尾方式时
- 控制节奏和密度时
- 安排章节分节时

**核心能力**:
- 5种结构模式的选择（时间三段式/哲学探索式等）
- 6种开篇方式的运用
- 6种结尾方式的运用
- 密度交替的控制

**输出**:
- 清晰的结构框架
- 有节奏感的段落安排
- 高密度与低密度的平衡

**何时不用**:
- 写作过程中（除非需要调整节奏）
- 修改具体语句时

➡️ [详见 skills/structure.md](skills/structure.md)

## 写作前检查清单（必读）

### 第一步：明确本篇定位

查看 `PROJECT_PLAN.md`，确认：
- [ ] 本篇在20篇中的位置（第几篇）
- [ ] 本篇在三幕结构中的位置（第一/二/三幕）
- [ ] 本篇的核心悖论是什么
- [ ] 本篇的风格层次（高密度/平衡/低密度）

### 第二步：检查跨篇联系

查看 `NETWORK.md`，确认：
- [ ] 需要承接哪些前篇（物品/人物/时间/地点）
- [ ] 需要为哪些后篇埋伏笔
- [ ] 涉及哪些已有细节（查"关键细节库"）
- [ ] 需要创造哪些新细节

### 第三步：选择结构模式

激活 **技能4: structure.md**，选择：
- [ ] 结构模式（时间三段式/哲学探索式/追寻叙事/对话体/元叙事）
- [ ] 开篇方式（意象/陈述/时间标记/对话/引用/元叙事）
- [ ] 预期的节奏控制（高-低密度交替）

### 第四步：设计虚构引言

必须包含：
- [ ] 作者名（虚构的中东/波斯学者）
- [ ] 书名（神秘主义特色）
- [ ] 年份（500-1500年）
- [ ] 国籍（古代王朝/帝国，如花剌子模、阿拔斯）
- [ ] 名句（与核心悖论呼应，1-2句话）

### 第五步：明确技能激活顺序

根据本篇内容，确定：
- [ ] 是否涉及世界观设定？→ 激活 **技能2**
- [ ] 是否需要理论工具？→ 激活 **技能3**
- [ ] 主导风格层次？→ 激活 **技能1**
- [ ] 结构框架？→ 激活 **技能4**

## 写作中实时检查

### 每写500字检查一次

**风格检查**（技能1）:
- [ ] 是否有精确的数字？
- [ ] 是否有"据说""传说""或者说"的不确定性？
- [ ] 情感是否冷静克制（无感叹号）？
- [ ] 句子长短是否交替？

**密度检查**（技能4）:
- [ ] 高密度段落是否连续超过3段？
- [ ] 是否需要低密度段落缓冲？
- [ ] 节奏是否有变化？

**设定检查**（技能2）:
- [ ] 阿尔法族是否只用"咿""呀"？
- [ ] 倾斜角度是否精确（0°/33°/90°/45°）？
- [ ] 道德守恒方程是否正确？

**理论检查**（技能3）:
- [ ] 理论是否服务于故事（不是炫技）？
- [ ] 是否展示而非解释理论？
- [ ] 理论是否导向悖论/无力感？

## 写作后完整检查

### A. 风格自检（技能1）

参考 `skills/style.md` 的自检清单：
- [ ] 简练性：无冗余形容词
- [ ] 精确性：有具体数字/地名/人名
- [ ] 冷峻性：无感叹号，情感隐藏
- [ ] 节奏感：短句长句交替
- [ ] 无力感：理性导向悖论

### B. 世界观检查（技能2）

如涉及，参考 `skills/worldbuilding.md`：
- [ ] 阿尔法族只用"咿""呀"，强调沉默
- [ ] 文字倾斜角度精确
- [ ] 两座城市的守恒方程正确
- [ ] 天书的定位悖论清晰

### C. 理论检查（技能3）

如涉及，参考 `skills/theory.md`：
- [ ] 理论是故事必需的（不是炫技）
- [ ] 理论被展示而非讲解
- [ ] 读者不懂理论也能理解故事
- [ ] 理论导向悖论/不可能
- [ ] 理论引入自然（通过人物/梦境/类比）

### D. 结构检查（技能4）

参考 `skills/structure.md`：
- [ ] 有明确的开篇（6种方法之一）
- [ ] 有逻辑的推进（不随意跳跃）
- [ ] 有开放的结尾（6种方法之一）
- [ ] 高密度段落不连续超过3段
- [ ] 核心悖论完整呈现

### E. 联系检查（NETWORK.md）

- [ ] 已植入的跨篇联系正确（细节一致）
- [ ] 已记录新创造的细节（更新"关键细节库"）
- [ ] 已标记预留的伏笔（更新"联系追踪表"）

## 技能组合策略（进阶）

### 场景1：写高密度哲学篇（如第9、10、14篇）

**技能激活顺序**:
1. **技能4**: 选择"哲学探索式"结构
2. **技能3**: 选定核心理论（罗素悖论/香农熵/维特根斯坦）
3. **技能1**: 使用"哲学思辨层次"（70%），少量诗性缓冲（20%）
4. **技能4**: 密度控制，每3段高密度后插入1段低密度

**注意**:
- 不要连续2篇高密度
- 理论必须文学化（故事优先）
- 保持冷峻，不说教

### 场景2：写低密度诗性篇（如第6、11、13、15篇）

**技能激活顺序**:
1. **技能4**: 选择"时间三段式"结构（清晨-黄昏-夜晚）
2. **技能1**: 使用"诗性散文层次"（60%），少量哲学（35%）
3. **技能1**: 意象开篇+意象结尾
4. **技能4**: 短句制造节奏，段落呼吸

**注意**:
- 避免意象重复（查已用过的）
- 诗性不等于抒情（依然冷峻）
- 依然需要核心悖论（但用意象展示）

### 场景3：写对话篇（如第7、18篇）

**技能激活顺序**:
1. **技能4**: 选择"对话体"结构
2. **技能1**: 对话简洁（每次发言不超过3句）
3. **技能3**: 理论通过对话展示（不同观点）
4. **技能1**: 最终双方都沉默（理性失败）

**注意**:
- 不用引号，用破折号或段落区分
- 对话不是为了达成共识，而是展示多元与徒劳
- 结尾必须开放（不是结论）

### 场景4：写元叙事篇（如第16、17、20篇）

**技能激活顺序**:
1. **技能4**: 选择"元叙事"结构
2. **技能1**: 使用"元叙事层次"（50%），配合哲学（30%）和诗性（20%）
3. **技能1**: 逐步打破第四堵墙（先正常叙事→转折→元层面）
4. **技能4**: 循环结尾（指向其他篇或回到开头）

**注意**:
- 元叙事不宜过早（第三幕才合适）
- 必须服务于主题（天书的不可把握）
- 保持博尔赫斯式克制（不滥用）

### 场景5：涉及阿尔法族的篇目（如第1、2、14篇）

**技能激活顺序**:
1. **技能2**: 检查阿尔法文字系统的使用规范
2. **技能2**: 确认只用"咿""呀"，强调沉默
3. **技能2**: 精确倾斜角度（0°/33°/90°/45°）
4. **技能1**: 用冷峻语言描述（不煽情）

**铁律**:
- 阿尔法人**绝不能说话**（只能"咿""呀"）
- 倾斜角度必须精确，有象征意义
- 强调视觉性、空间性、沉默

### 场景6：涉及两座城市的篇目（如第1、3、4篇）

**技能激活顺序**:
1. **技能2**: 检查道德守恒方程（W(x)+R(x)=K）
2. **技能2**: 确认索菲亚/卡珊德拉的具体运作
3. **技能1**: 展示而非解释守恒（通过故事）
4. **技能3**: 可选：引入守恒定律的数学/物理类比

**注意**:
- 索菲亚人不是"虚伪"，而是守恒的结果（不评判）
- 卡珊德拉人不是"善良"，而是守恒的另一端
- K₁≠K₂的悖论（第4篇揭示）

## 常见问题快速解决

### Q1: 我不确定本篇应该是哪种风格层次？

**决策流程**:
1. 查看 `PROJECT_PLAN.md` 的"20篇规划矩阵"
2. 看本篇的"风格层次"列
3. 看周围篇目（避免连续高密度）
4. 如果仍不确定，默认"平衡型"（50%哲学+40%诗性+10%元叙事）

### Q2: 我想引入一个理论，但不确定如何文学化？

**解决方案**:
1. 激活 **技能3: theory.md**
2. 查看"理论使用的原则"（文学优先，理论辅助）
3. 查看对应理论的"文学化表达"示例
4. 使用"理论引用的具体技巧"（借人物之口/梦境/类比/脚注）

### Q3: 我不知道如何开篇？

**解决方案**:
1. 激活 **技能4: structure.md**
2. 查看"开篇的六种方法"
3. 根据本篇风格层次选择：
   - 诗性篇→意象开篇
   - 哲学篇→陈述开篇或引用开篇
   - 对话篇→对话开篇
   - 元叙事篇→元叙事开篇
4. 参考已完成15篇的开篇

### Q4: 阿尔法族能不能说"你好"？

**答案**: **绝对不能**

**原因**:
- 激活 **技能2: worldbuilding.md**
- 查看"阿尔法族 → 基本规则 → 嗓音限制"
- 他们**只能**发"咿"(肯定/喜悦)和"呀"(否定/愤怒)
- 任何其他音节都**无法发出**
- 这是**不可违背的铁律**

### Q5: 我忘了某个细节（如索菲亚账簿的描述）？

**解决方案**:
1. 查看 `NETWORK.md`
2. 找到"关键细节库"
3. 搜索对应元素（如"索菲亚账簿"）
4. 使用精确相同的细节描述

### Q6: 这篇应该分几节/章？

**决策流程**:
1. 激活 **技能4: structure.md**
2. 查看"章节/分节的时机"
3. 规则：
   - 短篇（<5000字）→ 单章不分节，或用空行/***分隔
   - 中篇（5000-7000字）→ 10节左右
   - 长篇（>7000字）→ 多章或多节
4. 参考已完成15篇的分节

### Q7: 我想确保跨篇联系正确？

**检查流程**:
1. 查看 `NETWORK.md`
2. 找到"联系追踪表"
3. 确认"从篇目→到篇目"的联系
4. 查看"关键细节库"确保细节一致
5. 写作后更新"联系追踪表"和"关键细节库"

- **技能4（结构）**是框架，决定叙事的节奏

**四者缺一不可，四者完美融合，才是博尔赫斯式天书**。
